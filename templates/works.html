Step 1: Setup the project
$  roro background-removal ls
static
templates
background-removal.py
requirements.txt
web.py
roro.yml


Step 2: Compose the YAML file. The runtime sandbox contains all the infrastructure wiring, and it is customizable.
$  cat roro.yml
project: background-removal
runtime: rorodata/sandbox
services:
  - name: api
   function: background_removal.predict
  - name: default
   gunicorn -b 0.0.0.0:8080 webapp:app


Step 3: Just deploy and your application is live!
$  roro deploy
